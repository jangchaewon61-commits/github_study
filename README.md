# Chugumi.me - 추구미 기반 스타일 & 뷰티 큐레이션 웹 서비스

당신의 추구미(감성)을 찾고 스타일과 뷰티를 함께 큐레이션하세요.

## 🎨 프로젝트 개요

Chugumi.me는 이미지 기반 밸런스 게임을 통해 사용자의 추구미를 진단하고, 그에 맞는 패션 코디와 뷰티 제품을 한 번에 추천받을 수 있는 서비스입니다.

## ✨ 주요 기능

- **이미지 기반 진단**: 8개의 밸런스 게임 문항으로 정확한 추구미 진단
- **스타일 코디 추천**: 상의, 하의, 아우터가 어울리게 조합된 완벽한 코디
- **뷰티 제품 매칭**: 코디에 어울리는 메이크업 제품 추천
- **퍼스널 컬러 필터**: 쿨톤/웜톤 선택으로 커스터마이징
- **다양한 조합 제공**: 같은 추구미에서도 새로운 코디 계속 받기
- **원클릭 구매**: 각 상품으로 바로 쇼핑 페이지 이동

## 🎯 추구미 유형

1. **Pure_Minimal** (미니멀 시크) 🤍
2. **Y2K_Cyber** (사이버 Y2K) 💾
3. **Glow_Highteen** (글로우 하이틴) ✨
4. **Dark_Aesthetic** (다크 에스테틱) 🖤
5. **Romantic_Vintage** (로맨틱 빈티지) 🌸
6. **Sporty_Urban** (스포티 어반) 👟
7. **Luxury_Classic** (럭셔리 클래식) 👑
8. **Nature_Earthy** (내추럴 어스톤) 🌿

## 🛠 기술 스택

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Fonts**: Fredoka (귀여운 폰트), Outfit (디스플레이 폰트), Playfair Display (세련된 폰트)

## 🚀 설치 및 실행

### 설치

\`\`\`bash
npm install
\`\`\`

### 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

브라우저에서 \`http://localhost:3000\`을 열어주세요.
